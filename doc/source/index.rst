LibQi Documentation
===================

What is qi library?
-------------------

libqi is a shared library providing access to OS and system level
functionalities. You will find cross-platform methods to access the environment,
to start process, to open files with UTF-8 paths, to open shared libraries and
so on. Basic logging facilities are also provided. You will also find convenient
functions in :doc:`qi::path </api/namespaces/qipath>` namespace to access and
search files related to your project.
