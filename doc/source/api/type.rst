- :cpp:class:`qi::Type`

- :cpp:class:`qi::TypeInt`
- TypeFloat
- TypeString
- TypeList, TypeListIterator
- TypeMap, TypeMapIterator
- TypeTuple, TypeTupleIterator
- TypeBuffer,
- TypeDynamic
