GenericValue API
================

Overview
--------

The library provide generic container that can be used to manipulate values. Generic container wrap a Type* with a storage, it's a convenient class to manipulate Types.

- GenericInt
- GenericFloat
- GenericString
- GenericBuffer
- GenericList
- GenericMap
- GenericTuple
- GenericObject
- GenericValue
